# 🏏 IPL Analytics Dashboard

## 📌 Overview

This project analyzes IPL (Indian Premier League) data to extract insights about team performance, match trends, and player statistics using Python and Streamlit.

## 🚀 Features

* 📊 Team performance analysis
* 🏆 Top teams by wins visualization
* 🔍 Team-wise match analysis (interactive dropdown)
* 🎯 Toss impact analysis
* 🏏 Top players by runs

## 🛠️ Tech Stack

* Python
* Pandas
* Streamlit
* Matplotlib

## 📁 Dataset

* matches.csv (IPL match data)
* deliveries.csv (not uploaded due to size)

## ▶️ How to Run

```bash
pip install streamlit pandas
streamlit run app.py
```
