# 🏏 IPL Player Performance Analysis & Prediction

## Overview
An end-to-end data science project analyzing IPL cricket data 
from 2008-2022 using Python and Machine Learning.

## Key Findings
- 🏆 Mumbai Indians are the most successful IPL team (144 wins)
- 👑 Virat Kohli is the all-time top scorer (8,014 runs)
- 🎳 Yuzvendra Chahal is the top wicket taker (213 wickets)
- 🏟️ Eden Gardens has hosted the most matches (77)
- 🤯 Winning the toss barely matters — it's 50/50!

## Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn (Linear Regression, Random Forest)
- Jupyter Notebook

## ML Models Built
- Linear Regression — R2 Score: 0.99
- Random Forest — R2 Score: 0.99

## Dataset
Download the IPL dataset from Kaggle:
https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

## How to Run
1. Clone this repository
2. Download the dataset from Kaggle link above
3. Install requirements: pip install pandas numpy matplotlib seaborn scikit-learn jupyter
4. Open ipl_analysis.ipynb in Jupyter Notebook
5. Run all cells!
